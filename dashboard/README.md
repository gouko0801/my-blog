## 新規記事作成方法

/dashboard/content に新しくディレクトリを作成する
ディレクトリ名が記事のpathとなる

作成したディレクトリ内にindex.mdを作成する

下記フォーマットでindex.md内に記述を行う

```
---
title: "test2"
date: "2022/04/30"
tags: "test"
---
// 以下本文
俺は今日も在るだろうな
```
